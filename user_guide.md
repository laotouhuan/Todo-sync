# Todo 用户使用指南

欢迎使用 **Todo**！这是一款专为高效心流打造的极简跨平台待办事项管理工具，支持 **Windows 桌面端** 与 **Android 移动端** 双端无缝协同。

---

## 1. 软件简介

**Todo** 是一款追求极致简单与数据安全的跨平台待办事项管理工具。

* **双端协同**：Windows 桌面端提供深度沉浸的磨砂玻璃悬浮工作流，Android 移动端提供桌面小部件以便随时记录。
* **数据自主**：无中心化服务器，所有待办事项均本地保存在一个开放的 `todo_data.json` 文件中，通过您自己的坚果云云盘或 WebDAV 同步，确保数据绝对私密。

---

## 2. 同步设置与云端同步

多端协同依赖于两端读取同一个 `todo_data.json` 数据文件。建议优先配置 **坚果云 WebDAV 直连同步**：

> [!IMPORTANT]
> **多设备同步最佳实践**：
> 1. **首次初始化**：建议先使用 **Windows 桌面端**。在 Windows 端成功写入第一条待办数据后，手机端首次配置完成并保存设置时，**请务必手动点击一次“强制从云端拉取”**，以加载初始记录。
> 2. **多设备流转**：在不同设备（电脑与手机）之间切换使用时，请在使用前点击 **同步/刷新** 按钮，确保两端数据时刻保持最新。

### 2.1 坚果云 WebDAV 的账号与应用密码获取
1. 电脑登录 [坚果云官网](https://www.jianguoyun.com/)。
2. 点击右上角账户名，选择 **账户信息 -> 安全应用**。
3. 在“第三方应用管理”中点击 **添加应用**，输入名称（如 `todo`）并点击 **生成密码**。
4. **保存复制**系统生成的 16 位英文字符应用密码。

### 2.2 Windows 桌面端配置同步
1. 点击右上角 **设置 (齿轮图标)**。
2. 在“同步模式”中选择 **坚果云 WebDAV (直连)**。
3. 填入参数：
   * **服务器地址**：`https://dav.jianguoyun.com/dav/`
   * **账号**：您的坚果云注册邮箱
   * **应用密码**：步骤 2.1 生成的 16 位密码
   * **云端文件路径**：`我的坚果云/to-do/todo_data.json`
4. 点击 **保存**。

> [!NOTE]
> **本地同步文件夹模式**：如果您的 PC 安装了网盘客户端（如坚果云或 OneDrive），也可以选择 **本地坚果云目录** 并绑定网盘的本地文件夹。系统会自动检测本地文件变更并秒级刷新界面。

### 2.3 Android 移动端配置同步
1. 打开 Android 客户端，进入 **设置**。
2. 填写与 Windows 端完全一致的 WebDAV 参数（地址、账号、应用密码、云端文件路径）。
3. 保存后，点击“强制从云端拉取”对齐数据。

## 3. Windows 桌面端使用指南

Windows 客户端提供精致的磨砂玻璃悬浮窗，支持无边框拖拽，并可点击钉子图标 **固定置顶**。

### 3.1 待办管理与高级交互
* **添加与编辑**：直接在底部输入框输入任务并回车。点击待办文字可呼出“编辑”弹窗，用于细化截止日期、提醒时间、循环规则（每天/每周/每月）或拆分**子步骤/备注**。
* **快速结转**：在编辑弹窗中点击 **放到明天**，可一键将任务延期。
* **子步骤快捷复制**：在编辑弹窗的子任务区域，点击“复制内容”可一键将子步骤复制为 **纯文本**、**无序列表 ( - )** 或 **有序列表 ( 1. )** 发送给他人。
* **拖拽重排与快速改期**：在“今日聚焦”视图中，拖动未完成任务越过**已完成/未完成分割线**以上，即可将其快捷设为“今日任务”；拖到分割线以下，则清除日期变为“无日期”暂存任务。

### 3.2 💡 智能日期快捷输入（@ 语法）
在主界面底部输入框或闪电录入框中输入任务时，支持在**任务末尾**以 **`空格+@+时间标识`** 的格式快捷设置日期。系统会自动解析为截止日期并从内容中剥离该标识：

| 快捷输入示例 | 解析效果说明 |
| :--- | :--- |
| `写工作周报 @today` | 截止日期自动设为 **今天** |
| `准备演示 PPT @tomorrow` | 截止日期自动设为 **明天** |
| `背单词 50 个 @day` 或 `@daily` | 设为 **每天重复任务**（打卡后自动结转明天） |
| `每周读书笔记 @week` | 归类为 **本周打卡任务**（无目标次数） |
| `跑步锻炼 @week*3` | 设为 **本周打卡任务**，目标为 **3次** |
| `健身房打卡 @month*10` | 设为 **本月打卡任务**，目标为 **10次** |
| `提交财务审批 @06-20` | 自动设为 **当年 6 月 20 日** |
| `重要会议记录 @2026-07-15` | 设为 **指定年份和日期** |

### 3.3 快捷键与托盘交互
* **`Ctrl + Shift + Space`**：**全局闪电录入 (聚光灯模式)**。在系统任意界面下一键唤起悬浮框，输入任务并回车即可快速存入今日待办（按 `Esc` 退出）。
* **`Ctrl + Shift + T`**：一键显示/隐藏主窗口。
* **系统托盘**：左键单击托盘图标可显隐窗口，右键可选择退出。

## 4. Android 移动端使用指南

Android 原生客户端专为出行及碎片化时间设计。

### 4.1 桌面小部件 (App Widget)
强烈推荐在手机桌面上添加 Todo 桌面小部件，无需打开 App 即可享受极速体验：
1. 长按手机桌面空白处，选择 **添加小部件 / Widget** 并找到 **Todo**。
2. **小部件功能**：
   * **快捷查看**：直接在手机屏幕滚屏浏览今日到期的重点任务，以及本周/本月的打卡任务，并用优雅的下划线分隔。
   * **闪电新建**：点击小部件顶部的输入框，直接键入内容并回车保存为今日待办。
   * **快捷打勾**：直接在桌面点击待办（或打卡任务）前方的复选框，即可快速完成或记录一次打卡。

## 5. 数据安全与灾备恢复

为保证待办数据的万无一失，Todo 提供了多层数据安全防御和结转机制：

* **底层防损坏**：系统读写文件时采用**文件排他锁**，防止多进程冲突；并使用**原子级文件写入**（先写入 `.tmp` 临时文件，校验成功后瞬间重命名覆盖），防范因系统断电等造成文件损坏。
* **本地快照备份**：每当数据被修改，系统会在本地私有目录下自动创建一个带有时间戳的快照。系统会自动清理，**仅保留最新的 5 个备份**以节省空间。
* **一键恢复**：若发生误删或数据错乱，可进入 **设置 -> 数据恢复**，选择一个备份时间点一键还原。
* **上期任务导入**：在新的一周或一个月开始时，若有上一周期的未完成任务，系统会自动弹出导入窗口。导入时，**原任务的子步骤会被一并复制，并且子步骤的状态会自动重置为“未完成”**，方便新周期重新跟踪。

## 6. 自动检查更新 (OTA)

本应用内建了原生的更新检测与自动下载机制：

* **电脑端自动更新**：进入 **设置 -> 检查更新**，若有新版本，系统会自动下载最新的 Windows 安装包并在后台完成解压更新。
* **手机端覆盖安装**：进入 **设置 -> 检查更新**，若有新版本，应用内会直接下载最新版 `.apk` 并拉起覆盖安装，体验流畅无缝。
* **提示**：更新检测依赖于 GitHub 发布页，请确保您的网络能顺畅访问。
